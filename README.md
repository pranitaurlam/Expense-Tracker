A responsive Expense Tracker web application built using HTML, CSS, and JavaScript.
This app helps users track income and expenses, calculate balances, and manage spending with a simple and user-friendly interface.

Features
Login authentication (client-side)
Add income and expense transactions
Automatic balance calculation
Category-wise expense tracking
Delete transactions
LocalStorage data persistence
Dark mode toggle
Fully responsive design
